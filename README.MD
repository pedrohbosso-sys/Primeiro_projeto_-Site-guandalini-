# 🏛️ Guandalini Advogados - Website Institucional

Este é o meu primeiro projeto real, desenvolvido para o escritório **Guandalini Advogados**. O objetivo foi criar uma presença digital moderna, funcional e otimizada para a conversão de novos clientes através de uma interface intuitiva.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica para melhor SEO.
* **CSS3:** Estilização personalizada com foco em responsividade (Mobile First).
* **JavaScript (Vanilla):** Lógica de estados, animações de scroll e integração com APIs de contato.

## 🎯 Funcionalidades
- [x] Design Responsivo (Adaptável a celulares, tablets e desktops).
- [x] Botão flutuante de integração direta com WhatsApp.
- [x] Seções otimizadas para carregamento rápido.
- [x] Navegação Single Page para melhor experiência do usuário (UX).

## 🧠 Aprendizados
Durante este projeto, apliquei conceitos fundamentais de desenvolvimento web aprendidos no SENAI, como:
- Manipulação de DOM para elementos dinâmicos.
- Organização de código limpo (Clean Code).
- Implementação de layouts modernos sem dependência de frameworks pesados.
